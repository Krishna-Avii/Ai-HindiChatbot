Krishna AI — हिन्दी चैटबॉट (Hindi Diwas Special)
================================================

सेटअप
-----
1) Node 18+ आवश्यक।
2) प्रोजेक्ट रूट में `.env` बनाएँ और निम्न भरें:

```
OPENAI_API_KEY=your_key_here
PORT=3000
# MODEL=gpt-4o-mini
GOOGLE_API_KEY=
```

चलाएँ
-----
```
npm run dev
```
फिर खोलें: http://localhost:3000

स्क्रिप्ट्स
---------
```
npm run dev   // ऑटो-रिलोड सर्वर
npm start     // प्रोड रन
```

थीम विकल्प
-----------
- utsav: उत्सवी, सकारात्मक शैली
- aadhunik: आधुनिक, संवादात्मक शैली (डिफ़ॉल्ट)
- paramparik: परम्परागत, शुद्ध हिन्दी शैली
